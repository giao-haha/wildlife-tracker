Wildlife Sighting Tracker

Hi! I'm Gary, a CPA student at Seneca College. I enjoy web development, learning backend technologies, and building APIs with Express.js.

Project Description

This application tracks wildlife sightings and provides RESTful API endpoints to view, filter, and analyze observation data.

API Endpoints

Endpoint Description
/api/sightings All sightings
/api/sightings/verified Verified sightings
/api/sightings/species-list Unique species list
/api/sightings/habitat/forest Forest sightings
/api/sightings/search/eagle Search eagle
/api/sightings/find-index/moose Find moose index
/api/sightings/recent Recent 3 sightings
